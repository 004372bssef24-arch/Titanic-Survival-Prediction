# 🚢 Titanic Survival Prediction System

## Group 2 - Artificial Intelligence Project
### International Islamic University, Islamabad

---

## 📋 Project Overview

This project implements a complete machine learning system to predict passenger survival on the Titanic using **Logistic Regression**, **Random Forest**, and **Voting Ensemble** techniques. The system includes:

- Comprehensive Exploratory Data Analysis (EDA)
- Feature engineering and preprocessing pipeline
- Multiple ML models with hyperparameter tuning
- Interactive web application (GUI)
- Model comparison and evaluation

---

## 📊 Dataset

- **Source:** Kaggle Titanic Dataset
- **Samples:** 891 passengers
- **Features:** 12 attributes (age, gender, class, fare, etc.)
- **Target:** Survival (0 = No, 1 = Yes)

---

## 🎯 Key Findings

| Factor | Survival Rate |
|--------|---------------|
| Women | 74.0% |
| Men | 18.9% |
| 1st Class | 62.9% |
| 2nd Class | 47.3% |
| 3rd Class | 24.2% |
| Children (<12) | 59.8% |

---

## 🤖 Models Used

| Model | Accuracy | ROC-AUC |
|-------|----------|---------|
| Logistic Regression | 81.0% | 0.863 |
| Random Forest | 82.7% | 0.879 |
| Voting Ensemble | 82.1% | 0.874 |

**Best Model:** Random Forest (82.7% accuracy)
